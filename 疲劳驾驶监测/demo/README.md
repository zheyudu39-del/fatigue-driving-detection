# 疲劳驾驶检测系统 - Demo 版本

这是一个简化的 Demo 版本，用于快速体验疲劳驾驶检测的核心功能。

## 功能特点

- ✅ 实时人脸检测
- ✅ 眼睛闭合检测（EAR 算法）
- ✅ 打哈欠检测（MAR 算法）
- ✅ 疲劳警告提示
- ✅ 简洁的 Web 界面

## 快速开始

### 1. 安装依赖

```bash
pip install flask opencv-python mediapipe numpy
```

### 2. 启动 Demo

**Windows 用户**：双击 `启动Demo.bat`

**其他系统**：
```bash
python demo_app.py
```

### 3. 打开浏览器

访问 http://localhost:5001

点击「开始检测」按钮，允许浏览器访问摄像头。

## 体验提示

- 闭眼超过 1 秒会触发疲劳警告
- 张大嘴巴可以模拟打哈欠
- 确保面部在摄像头范围内
- 光线充足效果更好

## 与完整版的区别

Demo 版本简化了以下内容：
- 只保留规则模式（EAR/MAR 阈值判断）
- 移除了深度学习模型
- 移除了头部姿态检测
- 移除了阈值校准功能
- 界面更简洁

如需完整功能，请查看主项目。

## 技术栈

- Python 3.8+
- Flask（Web 服务）
- OpenCV（图像处理）
- MediaPipe（人脸关键点检测）
- NumPy（数值计算）
